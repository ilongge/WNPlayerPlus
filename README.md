## Installation

WNPlayer-ilongge is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'WNPlayer-ilongge', :git => 'https://gitee.com/ilongge/wnplayer-ilongge.git'
```
